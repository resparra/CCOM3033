Descripcion: Este programa le permite al usuario cargar un archivo con informacion
        de un paciente y organizar los diagnosticos de acuedo a la fecha.

Usuario: Al correr el programa se pide que entre el nombre del paciente, su apellido y el 
        numero de record. Del record del paciente se obtiene la fecha, el diagnostico y el 
        doctor que atendio al paciente. Estos records son organizados por fecha y guargado
	en un archivo con nombre <numero_de_record>.sorted.

Compilar:Para compilar el programa primero llegar al directorio donde se
        encuentra el Source File en este caso asigna6.cpp. Segundo se  compilar
        usando el comando g++ -c asigna6.cpp para crear el object file. Por
        ultimo usar el comando g++ -o Asig6 asigna6.o el cual crea un ejecutable
        con el Nombre Asig6, el cual se corre desde el terminal con su direccion.
